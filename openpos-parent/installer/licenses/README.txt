======= README FIRST =======

Thank you for installing meWaiter POS.

A mini micro friendly mobile POS for your waiters.

The fastest way to order from any mobile phone at your restaurant.

Using BitRock Installer the result should be:

1. Install RabbitMQ as a Background Service.
2. Jetty Embedded Server
3. Start.bat script which should start RabbitMQ, run Jetty with our .war deployed and run the swing OpenPOS app.
